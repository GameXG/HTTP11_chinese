以下核心规则包含在RFC RFC347的附录B.1中：ALPHA（字母），CR（回车），CRLF（CR LF），CTL（对照），DIGIT（十进制0-9） ，DQUOTE（双引号），HEXDIG（十六进制0-9 / AF / af），HTAB（水平制表符），LF（换行符），OCTET（任何8位数据序列），SP（空格）和VCHAR 任何可见的US-ASCII字符）。

> ```
